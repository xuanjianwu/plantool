# Usage for SGPlan: 
```python
>>> import sgplan
>>> argv = ['sgplan', '-o', 'domain.pddl', '-f', 'problem.pddl', '-out', 'output.soln']
>>> sgplan.run(argv)
```

## Instructions about arguments:

### OPTIONS   DESCRIPTIONS
```bash
-o  <str>    operator file name
-f  <str>    fact file name
-out <str>   output solution file name
